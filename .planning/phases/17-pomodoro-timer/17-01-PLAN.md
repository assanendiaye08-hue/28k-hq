       - Get tray icon via `TrayIcon.getById('main')` (cache the reference)
       - If remainingMs is null or <= 0, call `setTitle(null)` to clear
       - Otherwise format as `MM:SS` using `Math.floor(remainingMs / 60000)` and `Math.floor((remainingMs % 60000) / 1000).toString().padStart(2, '0')`
       - Call `setTitle(display)` and `setTooltip('28K HQ - ' + display)`

       - Define `SavedTimerState` interface matching store fields: phase, sessionId, phaseStartedAt, phaseDurationMs, totalWorkedMs, totalBreakMs, pomodoroCount, prePausePhase, pauseRemainingMs, plus all config fields (workDuration, breakDuration, longBreakDuration, longBreakInterval, targetSessions, autoStartBreak, autoStartWork, focus, goalId)
       - Export `saveTimerState(state: SavedTimerState)`: load store with `{ defaults: {} }`, set 'timerState' key, save
       - Export `loadTimerState(): Promise<SavedTimerState | null>`: load store, get 'timerState' key, return or null
       - Export `clearTimerState()`: load store, delete 'timerState' key, save

    3. **Timer audio** (`apps/desktop/src/lib/timer-audio.ts`):
       - Create a module-level `let alarmAudio: HTMLAudioElement | null = null`
       - Export `preloadAlarm()`: create `new Audio('/sounds/alarm-chime.mp3')`, set `preload = 'auto'`, play at volume 0 to warm up audio context (catches autoplay errors silently), assign to module var
       - Export `playAlarm()`: if alarmAudio exists, set `currentTime = 0`, call `play()` with catch for autoplay errors
       - NOTE: The actual sound file will be added later. For now the code handles missing file gracefully via catch.

    4. **Timer store** (`apps/desktop/src/stores/timer-store.ts`):
       - Use `create` from zustand (same pattern as auth-store, dashboard-store, goals-store)
       - Import `apiFetch` from `../api/client`, `TIMER_DEFAULTS` from `@28k/shared`, `saveTimerState`/`clearTimerState` from persistence, `updateTrayTitle` from tray helper
       - Define `TimerPhase = 'idle' | 'working' | 'on_break' | 'paused' | 'transition'`
       - State fields: phase, sessionId, phaseStartedAt, phaseDurationMs, totalWorkedMs, totalBreakMs, pomodoroCount, prePausePhase, pauseRemainingMs
       - Config fields: workDuration (default from TIMER_DEFAULTS.defaultWorkMinutes), breakDuration (default 5), longBreakDuration (default 15), longBreakInterval (default 4), targetSessions (null = unlimited), autoStartBreak (false), autoStartWork (false), focus (''), goalId (null)
       - `getRemainingMs()`: if paused return pauseRemainingMs; if phaseStartedAt is null return 0; otherwise `Math.max(0, phaseDurationMs - (Date.now() - phaseStartedAt))` (timestamp-based -- NEVER count ticks)
       - `isLongBreak()`: `longBreakInterval > 0 && pomodoroCount > 0 && pomodoroCount % longBreakInterval === 0`
       - `start(config)`: validate focus is non-empty (throw if empty). POST `/timer` with mode 'POMODORO', workDuration, breakDuration, focus, goalId, targetSessions, longBreakDuration, longBreakInterval. Set sessionId from response.id. Set phase='working', phaseStartedAt=Date.now(), phaseDurationMs=workDuration*60000, reset totals/pomodoroCount. Call saveTimerState.
       - `pause()`: calculate elapsed = Date.now() - phaseStartedAt. If phase was 'working', add elapsed to totalWorkedMs. If 'on_break', add elapsed to totalBreakMs. Set pauseRemainingMs = phaseDurationMs - elapsed. Set prePausePhase = current phase, phase = 'paused'. PATCH `/timer/${sessionId}` with action 'pause', remainingMs, totalWorkedMs, totalBreakMs, pomodoroCount, timerState: prePausePhase. Call saveTimerState. Call updateTrayTitle(null).
       - `resume()`: PATCH `/timer/${sessionId}` with action 'resume'. Set phaseStartedAt = Date.now(), phaseDurationMs = pauseRemainingMs, phase = prePausePhase, clear prePausePhase/pauseRemainingMs. Call saveTimerState.
       - `stop()`: calculate final totals (add any elapsed time from current running phase). PATCH `/timer/${sessionId}` with action 'stop', totalWorkedMs, totalBreakMs, pomodoroCount. Store the response (contains xpAwarded, leveledUp, newRank). Reset to idle. Call clearTimerState. Call updateTrayTitle(null). Return the stop response for UI display.
       - `completePhase()`: if phase is 'working': add elapsed to totalWorkedMs, increment pomodoroCount. Check if targetSessions reached (targetSessions !== null && pomodoroCount >= targetSessions). If reached, call stop(). If not, set phase='transition', determine next break duration (isLongBreak? longBreakDuration : breakDuration), set phaseDurationMs accordingly. If autoStartBreak, immediately transition to 'on_break' with new phaseStartedAt/phaseDurationMs. Otherwise stay in 'transition'. If phase is 'on_break': add elapsed to totalBreakMs. Set phase='transition'. If autoStartWork, immediately transition to 'working' with workDuration * 60000. Otherwise stay in 'transition'. Call saveTimerState.
       - `transitionToBreak()`: set phase='on_break', phaseStartedAt=Date.now(), phaseDurationMs = (isLongBreak ? longBreakDuration : breakDuration) * 60000. Call saveTimerState.
       - `transitionToWork()`: set phase='working', phaseStartedAt=Date.now(), phaseDurationMs=workDuration*60000. Call saveTimerState.
       - `restore(saved: SavedTimerState)`: restore all state fields from saved state. If phase was 'working' or 'on_break', validate phaseStartedAt is recent enough that remaining > 0. If remaining <= 0, call completePhase. Otherwise just restore.
       - `updateFocus(newFocus: string)`: set focus, PATCH `/timer/${sessionId}` with action 'pause' and focus (then immediately resume -- or just send focus update). Simpler: just update local state and persist, sync on next API call.

    5. **Timer tick hook** (`apps/desktop/src/hooks/use-timer-tick.ts`):

       - Return `{ remainingMs, formattedTime }` where formattedTime is the `MM:SS` string
       - useEffect with dependency on phase: if phase is 'working' or 'on_break', start a `setInterval(1000)`. On each tick: call getRemainingMs() for current value (timestamp-based, always accurate). Update remainingMs state. Call updateTrayTitle(remaining). If remaining <= 0, call completePhase(). Return cleanup function to clearInterval. If phase is not working/on_break, call updateTrayTitle(null) and return.
