    2. **Timer notifications helper** (`apps/desktop/src/lib/timer-notifications.ts`):
       - Import `getCurrentWindow` from `@tauri-apps/api/window`
       - Import `playAlarm` from `./timer-audio`
       - Import `sendNotification, isPermissionGranted, requestPermission` from `@tauri-apps/plugin-notification`
       - Export `async function onPhaseComplete(completedPhase: 'working' | 'on_break', focus: string)`:
         a. Play alarm: `playAlarm()`
         b. Bring window to foreground:
            ```
            const win = getCurrentWindow();
            await win.show();
            await win.unminimize();
            await win.setFocus();
            await win.requestUserAttention(2); // Critical on Windows
            ```
         c. Send OS notification as supplementary alert:
            ```
            let permitted = await isPermissionGranted();
            if (!permitted) { permitted = (await requestPermission()) === 'granted'; }
            if (permitted) {
              sendNotification({
                title: completedPhase === 'working' ? 'Work Complete!' : 'Break Over!',
                body: completedPhase === 'working'
                  ? `Nice work on "${focus}". Time for a break.`
                  : 'Ready to get back to work?',
              });
            }
            ```
       - Export `async function onSessionComplete(focus: string, xpAwarded: number)`:
         a. Same foreground behavior
         b. Notification: `title: 'Session Complete!'`, body: `"${focus}" — ${xpAwarded} XP earned!`
