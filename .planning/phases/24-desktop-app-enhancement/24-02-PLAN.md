Output: A new API endpoint returning active members and a compact UI indicator on the Today view.
</objective>

2. Create `apps/desktop/src/components/dashboard/GrindingIndicator.tsx`:
   - Read from `useActivityStore`.
   - Call `fetchGrinders()` in useEffect on mount. Also set up a 60-second polling interval (`setInterval`) to refresh -- clean up on unmount.
   - If `count === 0` and not loading: Show a subtle line "No one grinding right now" in text-text-tertiary text-xs. No Card wrapper needed -- keep it minimal.
   - If `count > 0`: Show a compact indicator:
     - A small green dot (w-2 h-2 rounded-full bg-success) pulsing with `animate-pulse`.
     - Text: "{count} grinding right now" in text-text-secondary text-sm.
     - Below: list up to 3 grinder names with their focus labels. Format: "Alex -- landing page" in text-xs text-text-tertiary. If more than 3, show "+N more".
   - NO Card wrapper for this component. It should feel like a status bar element, not a dashboard section. Use a simple div with subtle top border (border-t border-white/5 pt-3 mt-3).
