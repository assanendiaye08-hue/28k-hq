    **Discord OAuth helpers (apps/api/src/lib/discord-oauth.ts):**
    - Define `DISCORD_API = 'https://discord.com/api/v10'` and `TOKEN_URL = 'https://discord.com/api/oauth2/token'`
    - Define `TokenResponse` interface: `access_token, token_type, expires_in, refresh_token, scope`
    - Define `DiscordUser` interface: `id, username, avatar, global_name`
    - Export `async function exchangeCode(code, codeVerifier, redirectUri, clientId, clientSecret)`:
      - Build URLSearchParams with `grant_type: 'authorization_code'`, `code`, `redirect_uri`, `client_id`, `client_secret`, `code_verifier`
      - POST to TOKEN_URL with `Content-Type: application/x-www-form-urlencoded`
      - Return parsed JSON as `TokenResponse`
      - On non-ok response, throw Error with status and body text
    - Export `async function getDiscordUser(accessToken)`:
      - GET `DISCORD_API/users/@me` with `Authorization: Bearer {accessToken}`
      - Return parsed JSON as `DiscordUser`

    **Auth routes (apps/api/src/routes/auth.ts):**
    Export a Fastify plugin (async function, NOT wrapped with fp -- route plugins use encapsulation) with 3 routes:

    1. `POST /discord` (rate-limited: `{ max: 5, timeWindow: '1 minute' }` via route config):
       - Accept body `{ code: string, codeVerifier: string, redirectUri: string }`
       - Validate with Zod: all three fields required, strings, non-empty
       - Call `exchangeCode()` with config.DISCORD_CLIENT_ID and config.DISCORD_CLIENT_SECRET
       - Call `getDiscordUser()` with the returned access_token
       - Lookup `fastify.db.discordAccount.findUnique({ where: { discordId: discordUser.id }, include: { member: true } })`
       - If no account found, return 403: `{ error: 'Not a 28K HQ member' }`
       - Sign JWT access token: `fastify.jwt.sign({ sub: account.memberId, did: discordUser.id }, { expiresIn: '15m' })`
       - Generate refresh token: `randomBytes(32).toString('hex')` from `node:crypto`
       - Create RefreshToken record: `fastify.db.refreshToken.create({ data: { memberId: account.memberId, token: refreshTokenValue, expiresAt: 30-day-from-now } })`
       - Return 200: `{ accessToken, refreshToken: refreshTokenValue, member: { id, displayName, discordId, avatar } }`

    2. `POST /refresh` (rate-limited: `{ max: 10, timeWindow: '1 minute' }`):
       - Accept body `{ refreshToken: string }`
       - Lookup `fastify.db.refreshToken.findUnique({ where: { token: refreshToken } })`
       - If not found or expired (`expiresAt < now`), return 401: `{ error: 'Invalid or expired refresh token' }`
       - Delete the used refresh token (rotation): `fastify.db.refreshToken.delete({ where: { id: found.id } })`
       - Create new refresh token with fresh 30-day expiry
       - Sign new JWT access token with same payload (`sub: memberId, did: discordId` -- look up member's discordId)
       - Get discordId: `fastify.db.discordAccount.findFirst({ where: { memberId: found.memberId } })`
       - Return 200: `{ accessToken, refreshToken: newRefreshTokenValue }`

    3. `POST /logout`:
       - Requires authentication (add `preHandler: [authenticate]`)
       - Accept body `{ refreshToken: string }` (optional -- also delete all for this member if not provided)
       - If refreshToken provided: `fastify.db.refreshToken.deleteMany({ where: { token: refreshToken } })`
       - If not provided: `fastify.db.refreshToken.deleteMany({ where: { memberId: request.memberId } })` (logout all sessions)
       - Return 200: `{ success: true }`
