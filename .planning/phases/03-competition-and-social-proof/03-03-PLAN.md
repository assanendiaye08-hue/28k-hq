Build the seasonal system with auto-bootstrapping Season 1, 2-month season cycles, end-of-season archival with snapshots, #hall-of-fame channel with permanent season summaries, temporary champion roles, and the /season command for viewing past seasons.

The hall-of-fame provides permanent bragging rights while the temporary champion role gives immediate recognition without creating lasting hierarchy.

    `src/modules/season/manager.ts`:
    Season lifecycle management. All functions accept ExtendedPrismaClient.

    1. `getActiveSeason(db)`: Query Season where active = true. Return Season or null.

    2. `getSeasonByNumber(db, number)`: Query Season where number = N. Return Season with snapshots included.

    3. `bootstrapSeason(db, logger)`: Called on bot ready if no active season exists.
       - Create Season { number: 1, startedAt: now, active: true }.
       - Log: "Season 1 started (auto-bootstrap)".
       - Return the created season.

    4. `endSeason(db, client, events, logger)`: The core season transition function. Must be atomic-safe.
       - Get active season. If none, warn and return.
       - Import calculator functions from leaderboard module (getXPLeaderboard, getVoiceLeaderboard, getStreakLeaderboard).
       - **Step 1: Snapshot.** Query all three leaderboards for the season's date range (seasonStart to now). For each dimension ("xp", "voice", "streaks"), create SeasonSnapshot records for every member with their position and value. Use createMany for batch insert.
       - **Step 2: Close season.** Update the active season: set endedAt = now, active = false.
       - **Step 3: Champion role.** Find the #1 position in XP leaderboard snapshot. Create a Discord role "Season {N} Champion" with gold color, positioned above rank roles (use guild.roles.create with hoist: true). Assign it to the champion member's Discord account(s). Store the role ID and removal date in BotConfig (key: `season_{N}_champion_role_id`, `season_{N}_champion_role_expires`).
       - **Step 4: Post hall-of-fame.** Call postSeasonSummary (from hall-of-fame.ts) with the season data and snapshots.
       - **Step 5: Start new season.** Create Season { number: N+1, startedAt: now, active: true }.
       - **Step 6: Emit events.** Emit seasonEnded(oldNumber) and seasonStarted(newNumber).
       - Log: "Season {N} ended. Season {N+1} started. Champion: {displayName}"

    5. `checkSeasonExpiry(db, client, events, logger)`: Called by daily cron.
       - Get active season. If none, call bootstrapSeason.
       - Calculate days elapsed since season startedAt.
       - If days >= SEASON_DURATION_DAYS, call endSeason.
       - Also check for expired champion roles: query BotConfig for champion_role_expires entries. If any are past due, find and delete the Discord role, remove the BotConfig entries. Log: "Removed Season {N} Champion role (expired)".

    6. `getSeasonSummary(db, seasonNumber)`: For the /season command.
       - Get season by number (with snapshots).
       - Group snapshots by dimension.
       - Return structured data: { season, xpRankings, voiceRankings, streakRankings }.

    **Important -- what resets vs carries over (per research discretion recommendation):**
    - Resets: Leaderboard positions are computed from data within the season's date range. No data is deleted.
    - Carries over: Member.totalXp (lifetime), Member.currentStreak, Member.longestStreak, all XPTransaction records, all VoiceSession records.
    - This means there is NO destructive "reset" operation. Seasonal rankings are just date-range filtered queries. Season 1 data is always available.

       - Find #hall-of-fame channel by name in the guild. If it doesn't exist, create it under "The Hub" category with read-only permissions (everyone can read, only bot can send).
       - Return the channel.

    2. `postSeasonSummary(client, db, season, snapshots)`:
       - Build a rich embed for the season summary:
         - Title: "Season {number} -- Final Standings"
         - Color: CHAMPION_ROLE_COLOR (gold)
         - Thumbnail: trophy emoji or server icon
         - Fields:
           - "XP Champion": #1 from xp snapshots with their total
           - "Voice Champion": #1 from voice snapshots with hours formatted
           - "Streak Champion": #1 from streaks snapshots with days
           - "Top 5 XP": Numbered list of top 5
           - "Top 5 Voice Hours": Numbered list of top 5
           - "Top 5 Streaks": Numbered list of top 5
         - Footer: "Season ran: {startedAt formatted} -- {endedAt formatted} ({SEASON_DURATION_DAYS} days)"
       - Send to #hall-of-fame channel.
       - Pin the message (channel.messages.pin) -- permanent bragging rights.
